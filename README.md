# Katina’s Kafé ☕️
> **A Cozy Sanctuary in Kigali, Rwanda**

Welcome to the digital home of **Katina’s Kafé**—an elegant space designed for connection, artisan coffee, inclusive experiences, and community atmosphere in Kigali, Rwanda. 

This modern, highly responsive web application features a fully editorial design, fluid animations, an immersive virtual menu, an inline table reservation concierge, and unique storytelling sections.

---

## ✨ Features & Highlights

- **🍂 Editorial Minimalist Aesthetic**: A beautiful Swiss-style layout styled with a customizable dark Cosmic Slate theme, generous negative space, deliberate typographic tracking, and high-contrast styling.
- **📱 Inclusive Design ("Sign of the Day")**: Highlighting our program where guests can learn basic sign language while ordering, bridging worlds one cup at a time.
- **✨ Active Hero Experience**: Highlighting dynamic animated scroll indicators and local coordinates of Kigali (`1° 57' 01" S, 30° 03' 40" E`).
- **📖 Immersive Digital Menu**: Easily browse our gourmet selections and specialty offerings categorized beautifully across beverages, pastries, and dining options.
- **📅 Table Reservation Concierge**: A seamless inline booking wizard for securing your spot at our Kigali bar.
- **🎬 Interactive Glassmorphic Media**: An immersive embedded atmospheric background video that visitors can control at the tap of a customized play/pause button.
- **💬 Feedback & Engagement**: A direct channel for guests to leave notes and get in touch with the team.

## 🚀 Getting Started

Follow these instructions to run the project locally.

### 1. Prerequisites

Make sure you have Node.js and npm installed on your system.
- **Node.js**: `v18.x` or higher
- **npm**: `v9.x` or higher

### 2. Installation

Clone this repository or extract the project zip, navigate into the root directory, and install the dependencies:

```bash
npm install
```

### 3. Development Server

Start the development server with Hot Module Replacement (HMR) disabled/enabled at port `3000`:

```bash
npm run dev
```

Your app will be available at [http://localhost:3000](http://localhost:3000).

### 4. Build for Production

Compile the production-ready assets into the `dist/` folder:

```bash
npm run build
```

## 🎨 Design Philosophy

Katina's Kafé uses custom grid bento architectures, subtle line dividers, dot leading alignments, and micro-interactive elements designed to support inclusive gestures. Visual hierarchies are prioritized without relying on boilerplate cards, allowing the user to experience the space's elegant mood before visiting.
